## 🧠 The MIT-Style Way to Learn Programming & AI

MIT, Harvard CS50, and all top CS departments follow the **same core learning pattern**. It's not magic — it's a system. Here's how it works:

| Step | What It Is | What You Do |
|------|-----------|-------------|
| 1️⃣ | **Concept First** | Short lecture or reading to introduce an idea |
| 2️⃣ | **Guided Examples** | The teacher solves problems in front of you |
| 3️⃣ | **Problem Sets** | You struggle on your own — homework, labs, challenges |
| 4️⃣ | **Office Hours** | Q&A, debugging, clarification when you're stuck |
| 5️⃣ | **Projects & Exams** | Build something non-trivial, test under constraints |

### 🔁 How to Copy This With YouTube + Free Platforms

```
Lectures        →  CS50, MIT OCW, freeCodeCamp, Stanford courses
Guided Examples →  Tech With Tim, Corey Schafer, CodeWithHarry, Krish Naik
Problem Sets    →  CS50/OCW assignments, LeetCode, Codeforces, Kaggle
Office Hours    →  AI chat (for hints, not full solutions) + Discord forums
Projects        →  freeCodeCamp projects, Kaggle competitions, your GitHub apps
```

> 💡 **Key mindset**: Don't just *watch videos* — build your own mini-MIT semester from them. Every channel below maps to a course. Treat it like one.

## 💻 Core CS & Programming Channels (Global Backbone)

These channels form the **spine of your CS education**. Treat their playlists like official university courses.

---

### 🔵 [freeCodeCamp.org](https://www.youtube.com/@freecodecamp)

- Full-length courses (4–12 hours) on **C, C++, Python, JavaScript, React, Node, SQL, Data Structures, and Machine Learning**
- Many "university-style" videos: Data Structures in C++, Full Python Course, ML with Python
- CS50 mirrored with full permission from Harvard
- 🏷️ *Best for: Complete beginner to intermediate developer*

---

### 🔴 [Harvard CS50](https://www.youtube.com/@cs50)

- The original CS50 channel — full **CS50** (C + Algorithms + Web), **CS50 AI**, **CS50 Web**, and Shorts
- True MIT-style structure: lectures + problem sets + labs are all **100% free** at [cs50.harvard.edu](https://cs50.harvard.edu)
- One of the most complete free CS courses ever made
- 🏷️ *Best for: Foundational CS with real problem sets and structure*

---

### 🟠 [MIT OpenCourseWare](https://www.youtube.com/@mitocw)

- Official MIT lectures on: **Introduction to CS (6.0001/6.0002), Algorithms (6.006), Operating Systems, Compilers, AI**, and more
- Use [ocw.mit.edu](https://ocw.mit.edu) for assignments and exams — YouTube for the actual lectures
- This is as close to sitting in an MIT classroom as it gets — for free

### 🟡 [Stanford Online / Stanford University](https://www.youtube.com/@stanfordonline)

- **CS106A/B, CS224N (NLP), CS229 (Machine Learning), CS231n (CNNs)** and fresh LLM seminars

## 🏗️ How to Build "Everything" Step by Step (Sample Roadmap)

Here's how you turn this channel list into a **1–2 year MIT-style self-degree**. This isn't a rough plan — follow it with discipline and you'll come out the other side as a serious engineer.

---

### 🔷 Semester 1 — Programming & CS Foundations

**Goal: Write real code. Understand what a computer actually does.**

- 📺 **CS50** (YouTube + official site) + **freeCodeCamp** Python / JS course
- 📝 Weekly: finish lectures → complete all problem sets → solve 3–5 LeetCode Easy problems
- 🛠️ Build: 1 small project per month (a calculator, a to-do app, a basic CLI tool)

```
Channels:  Harvard CS50, freeCodeCamp, Tech With Tim, Programming with Mosh
Platforms: cs50.harvard.edu, leetcode.com (Easy)
```

---

### 🔷 Semester 2 — Data Structures, Algorithms & Web Basics

**Goal: Solve problems systematically. Build your first web presence.**

- 📺 **takeUforward / Love Babbar / Apna College** DSA playlist
- 📺 **Traversy Media / Net Ninja** for basic web development
- 📝 LeetCode Easy → Medium: target **200+ problems** over the semester
- 🛠️ Build: a personal portfolio website + 2 DSA visualizer tools

```
Channels:  takeUforward (Striver), Apna College, Traversy Media, The Net Ninja
Platforms: leetcode.com, codeforces.com, freecodecamp.org
```

---

### 🔷 Semester 3 — Math + Introduction to ML

**Goal: Stop being afraid of math. Build your first ML models.**

- 📺 **3Blue1Brown** linear algebra + **StatQuest** statistics basics
- 📺 **DeepLearning.AI** "AI for Everyone" + introductory ML playlist
- 📝 Build **2–3 ML mini-projects**: regression, classification, EDA
- 🛠️ Push everything to GitHub with proper READMEs

```
Channels:  3Blue1Brown, StatQuest, DeepLearning.AI, Data Professor, Sentdex
Platforms: kaggle.com, freecodecamp.org (ML certification)
```

---

### 🔷 Semester 4 — Deep Learning & LLMs

**Goal: Understand how neural networks actually work. Deploy something AI-powered.**

- 📺 **Andrej Karpathy's "Neural Networks: Zero to Hero"** (mandatory — watch every second)
- 📺 **DeepLearning.AI LLM series** + Nicholas Renotte / Sentdex for applied projects
- 📝 Build: vision project, basic NLP pipeline, a simple RAG system
- 🛠️ Deploy at least **one real AI application** — live URL, clean docs, GitHub repo

```
Channels:  Andrej Karpathy, DeepLearning.AI, Nicholas Renotte, Hugging Face, AssemblyAI
Platforms: huggingface.co, kaggle.com (competitions), cs50.harvard.edu/ai
```

---

### 🔷 Semester 5 — Systems, Projects & Interviews

**Goal: Go deep into how systems work. Build a capstone. Get interview-ready.**

- 📺 **MIT OCW** for OS/Networks/Systems + **Low Level Learning / Computerphile** for intuition
- 📝 Daily LeetCode + mock interviews using **takeUforward / Scaler** content
- 🛠️ Capstone: **fullstack + AI**, deployed, with documentation and a README that looks like serious open-source work
- 🏁 Final milestone: 400+ LeetCode problems solved, 1 deployed fullstack AI app, GitHub portfolio complete

```
Channels:  MIT OCW, Low Level Learning, Computerphile, Scaler, Ben Eater
Platforms: ocw.mit.edu (assignments/exams), leetcode.com, kaggle.com
```
