> *YouTube is not the problem. Passive consumption is.*

This repository exists for one reason: to turn the best free CS and AI education on the internet into a **structured, semester-based learning system** — the kind that produces engineers who build things, not people who "know a lot of stuff."

You get two things here:

- 📺 A refined, working list of **60+ YouTube channels and free platforms** consistently at the top of CS/AI learning in 2025–2026
- 🏛️ An **MIT-style learning framework** — how to use these channels like real university courses, with lectures, problem sets, projects, and self-assessment — so you come out the other side able to build serious systems

## 🧠 The MIT-Style Learning Framework

MIT, Harvard CS50, and every serious CS program follow the same core pattern. It is not magic — it is a system. Copy it exactly.

| # | Step | What It Is | What You Do |
|---|------|-----------|-------------|
| 1️⃣ | **Concept First** | Introduce the idea | Short lecture or reading before anything else |
| 2️⃣ | **Guided Examples** | Watch problems being solved | Follow every step — pause, rewind, redo |
| 3️⃣ | **Problem Sets** | Struggle alone | No shortcuts, no skipping, no asking until you've tried |
| 4️⃣ | **Office Hours** | Get unstuck | Debugging, Q&A, clarification — only when genuinely stuck |
| 5️⃣ | **Projects & Exams** | Prove it works | Build something non-trivial under real constraints |

### 🔁 Mapping This to Free Resources

```
📖 Lectures        →  CS50, MIT OCW, Stanford, DeepLearning.AI
🔍 Guided Examples →  Tech With Tim, Corey Schafer, Karpathy, Krish Naik
📝 Problem Sets    →  CS50/OCW assignments, LeetCode, Codeforces, Kaggle
💬 Office Hours    →  AI chat (hints only, never solutions) + Discord forums
🛠️ Projects        →  freeCodeCamp projects, Kaggle competitions, your GitHub
```

> **Core principle:** Don't just watch — build your own mini-MIT semester from every playlist. Treat each channel like a course with a syllabus. The discipline gap between watching and building is where most people stop. **Don't stop there.**

## 💻 Core CS & Programming Channels

The structural backbone of your CS education. Treat every major playlist as an official course with a start date and a completion deadline.

---

### 🟠 freeCodeCamp.org
Full-length courses (4–12 hours each) covering C, C++, Python, JavaScript, React, Node, SQL, Data Structures, and Machine Learning. Includes CS50 content mirrored with full permission from Harvard.  
🏷️ **Best for:** Complete beginner through intermediate developer

---

### 🔴 Harvard CS50
The gold standard of free CS education. Full CS50 (C + Algorithms + Web), CS50 AI, CS50 Web, and Shorts — all with real problem sets and labs at [cs50.harvard.edu](https://cs50.harvard.edu). This is what a real university course looks like.  
🏷️ **Best for:** Foundational CS with genuine problem sets and academic structure

---

### 🟡 MIT OpenCourseWare
Official MIT lectures: Introduction to CS (6.0001/6.0002), Algorithms (6.006), Operating Systems, Compilers, AI, and more. Pair with [ocw.mit.edu](https://ocw.mit.edu) assignments and exams. As close to an MIT classroom as the internet allows.  
🏷️ **Best for:** Theory-heavy CS with math-backed rigor

---

### 🔵 Stanford Online / Stanford University
CS106A/B, CS224N (NLP), CS229 (ML), CS231n (CNNs), and current LLM seminars. The natural second step after CS50 and MIT fundamentals.  
🏷️ **Best for:** ML specialization and AI research-level content

---

### 🟢 Traversy Media
Modern web development: HTML, CSS, JavaScript, Node, Express, React, Next.js, MongoDB. Crash courses consistently among the best quick entry points for any web stack.  
🏷️ **Best for:** Job-ready full-stack web skills

---

### 🔵 The Net Ninja
Exceptionally organized playlists on JavaScript, React, Node, TypeScript, Tailwind, Firebase, and full application clones. Among the cleanest playlist structures on YouTube.  
🏷️ **Best for:** Frontend and full-stack focused learners

---

### 🟣 Programming with Mosh
High-quality structured introductions to Python, Java, C#, OOP, and clean code practices. Strong for building fundamentals correctly from the beginning.  
🏷️ **Best for:** Beginner to intermediate — proper OOP foundations

---

### 🐍 Corey Schafer
Deep Python: OOP, virtual environments, Flask, Django, Pandas, Git, Docker. One of the most respected Python educators on the platform.  
🏷️ **Best for:** Intermediate Python and backend development

---

### 🟩 Tech With Tim
Python basics, Pygame, automation, and beginner AI/ML. Approachable but consistently practical — a strong ramp into Python project work.  
🏷️ **Best for:** Beginners building their first Python projects

---

### 🔷 CS Dojo
Python tutorials with intro data structures and interview preparation. Clean, deliberate explanations for early-stage learners.  
🏷️ **Best for:** Beginners transitioning into DSA

---

### ⚙️ The Cherno
C++ from scratch through game engine architecture, memory management, and graphics. Essential for understanding what actually happens beneath your Python.  
🏷️ **Best for:** Low-level programming, C++, engine and systems development

---

### 🌐 Web Dev Simplified
Modern JavaScript, React, APIs, and performance — direct, no filler, focused on what matters for serious front-end work.  
🏷️ **Best for:** JavaScript-first developers

---

### 🎨 Kevin Powell
CSS and design fundamentals, taught properly. One of the few channels that genuinely *explains* CSS rather than just demonstrating it.  
🏷️ **Best for:** Front-end developers who want production-quality UI

---

### 📘 Academind
React, Node, Next.js, TypeScript, and full-stack courses with tight structure and clear diagrams. Max Schwarzmüller's teaching is thorough and well-paced.  
🏷️ **Best for:** Full-stack developers at intermediate level

---

### 🔧 Engineer Man
Short, sharp videos on performance, C, scripting, and tooling. Builds the mindset of engineering code — not just writing it.  
🏷️ **Best for:** Systems thinking and engineering perspective

---

### 🏢 Google for Developers & Microsoft Developer
Official content on web standards, Android, .NET, GitHub, Azure, and AI tooling. Critical for staying aligned with platform-specific APIs.  
🏷️ **Best for:** Platform-specific development and official tooling

## 📐 Math, CS Theory & Systems

You cannot think like a CS/AI engineer on tutorials alone. These channels are your visual textbooks for the hard material that separates builders from script-followers.

---

### 🔵 3Blue1Brown
Essence of Linear Algebra, Essence of Calculus, and a landmark neural networks series. Arguably the single best mathematical intuition channel on the internet. The visuals alone are worth every minute.  
🏷️ **Best for:** Building genuine mathematical intuition for AI

---

### 📊 StatQuest with Josh Starmer
Statistics and ML algorithms explained with deliberate simplicity: regression, decision trees, model evaluation, Bayesian thinking. Essential before moving into ML work.  
🏷️ **Best for:** Statistics for machine learning — accessible and thorough

---

### 📏 Khan Academy
Complete sequences on algebra, calculus, probability, and statistics. The safety net — close any math gap you encounter while following MIT or Stanford content.  
🏷️ **Best for:** Filling foundational math gaps at any point in the roadmap

---

### 💾 Computerphile
Short concept videos on hashing, cryptography, protocols, file systems, OS scheduling, and Big-O. High-quality CS theory in 10–15 minute bites.  
🏷️ **Best for:** CS concepts as digestible, watchable theory

---

### 🔌 Ben Eater
Build an 8-bit computer from scratch on breadboards — transistors, logic gates, all of it. Truly understand how CPU, RAM, buses, and clocks work at the hardware level.  
🏷️ **Best for:** Computer architecture from absolute first principles

---

### 🖥️ Low Level Learning
OS internals, networks, C, security, and low-level debugging explained with clarity. An excellent companion to MIT's systems courses.  
🏷️ **Best for:** Systems programming, OS internals, security fundamentals

---

### 🔢 Numberphile
Curious, well-made math content — keeps you comfortable with proofs and abstract ideas throughout the degree.  
🏷️ **Best for:** Staying mathematically curious when theory gets heavy

## 🤖 AI, ML, Data Science & LLMs

Once you can code and follow foundational math, move into this tier. This is where the frontier actually is.

---

### 🧪 DeepLearning.AI
Andrew Ng and team: deep learning, LLMs, GenAI, RAG, agents, and practical ML workflows. The most structured progressive pathway into production-grade AI on YouTube.  
🏷️ **Best for:** Structured, progressive deep learning education

---

### ⚡ Andrej Karpathy
*Neural Networks: Zero to Hero* — transformer breakdowns, training LLMs from scratch in PyTorch, and some of the most important AI education ever put on video. **Non-negotiable.**  
🏷️ **Best for:** Understanding how LLMs actually work, from the inside out

---

### 🐍 Sentdex
Applied ML, reinforcement learning, computer vision, and real AI projects in Python. Project-based series for builders who need to ship, not just study.  
🏷️ **Best for:** Applied AI with Python — RL, CV, real projects

---

### 📄 Two Minute Papers
Accessible summaries of the latest AI research. Essential for staying current without reading 40-page papers daily.  
🏷️ **Best for:** Keeping pace with AI research in digestible form

---

### 🔬 Yannic Kilcher
Deep dives into specific papers and model architectures. Best for seriously parsing a new idea from first principles.  
🏷️ **Best for:** Research-level understanding of papers and architectures

---

### ☕ AI Coffee Break with Letitia
Conceptual explainers on AI topics with strong visualizations: attention, diffusion, embeddings, and more. A clean entry point that doesn't intimidate.  
🏷️ **Best for:** Conceptual AI explainers without the noise

---

### 🏛️ Henry AI Labs
LLM architectures, RLHF, alignment, and scaling laws. For understanding the decisions behind modern AI systems at a design level.  
🏷️ **Best for:** LLM internals, alignment thinking, and scaling

---

### 📈 Data School & Data Professor
End-to-end data science workflows in Python — from exploration through deployment. Grounded in real business data and real pipelines.  
🏷️ **Best for:** Practical data science from raw data to production

---

### 👁️ Nicholas Renotte
Vision, agents, and complete AI application builds — object detection, chatbots, and more. Hands-on work that produces a real, deployable output.  
🏷️ **Best for:** Applied AI projects and end-to-end builds

### 🎙️ AssemblyAI
Clear transformer and attention explainers, speech models, and SOTA breakdowns. Production-quality educational content from an actual AI company.  
🏷️ **Best for:** Transformers, NLP pipelines, and speech AI

### 🏆 Kaggle (YouTube)
Competitions, tutorials, and winning notebook walkthroughs. Real problems, real data, real benchmarks — the most honest exam you can give yourself in ML.  
🏷️ **Best for:** Applying ML skills to competitive, real-world problems

---

### 🤗 Hugging Face (YouTube)
Transformers, diffusion, RL, and open-source tooling — often with live coding. The official voice of the open-source AI ecosystem.  
🏷️ **Best for:** Modern AI tooling, open-source models, and the HF ecosystem

---

### 🔭 OpenAI | Anthropic | Google DeepMind
Official talks, demos, and research keynotes from the frontier labs. Essential for understanding where the field is heading and why.  
🏷️ **Best for:** Staying current with frontier AI research and direction

## 🗺️ 5-Semester Roadmap

This is how you convert the channel list into a **1–2 year MIT-style self-degree**. Follow it with the same discipline you'd give a paid program — because the only difference between this and a paid program is that the accountability is entirely yours.

---

### 🔷 Semester 1 — Programming & CS Foundations
**🎯 Goal:** Write real code. Understand what a computer actually does at a foundational level.

- 📺 **Lectures:** CS50 (YouTube + official site) + freeCodeCamp Python/JS
- 📝 **Problem sets:** Complete all CS50 problem sets — every single one
- 💪 **Weekly practice:** 3–5 LeetCode Easy problems per week
- 🛠️ **Builds:** 1 small project per month — calculator, CLI tool, to-do app
- **Channels:** Harvard CS50 · freeCodeCamp · Tech With Tim · Programming with Mosh
- **Platforms:** [cs50.harvard.edu](https://cs50.harvard.edu) · [leetcode.com](https://leetcode.com)

---

### 🔷 Semester 2 — Data Structures, Algorithms & Web Basics
**🎯 Goal:** Solve problems systematically. Build your first deployable web presence.

- 📺 **Lectures:** takeUforward / Apna College DSA + Traversy Media / Net Ninja for web
- 📝 **Problem sets:** LeetCode Easy → Medium — targeting 200+ problems over the semester
- 🛠️ **Builds:** Personal portfolio site + 2 DSA visualizer tools
- **Channels:** takeUforward · Apna College · Traversy Media · The Net Ninja
- **Platforms:** [leetcode.com](https://leetcode.com) · [codeforces.com](https://codeforces.com) · [freecodecamp.org](https://freecodecamp.org)

---

### 🔷 Semester 3 — Mathematics & Introduction to ML
**🎯 Goal:** Stop being afraid of math. Build your first working ML models.

- 📺 **Lectures:** 3Blue1Brown linear algebra + calculus + StatQuest statistics
- 🔍 **Guided examples:** DeepLearning.AI intro ML series + Data Professor
- 🛠️ **Builds:** 2–3 ML mini-projects: regression, classification, EDA — all pushed to GitHub with proper READMEs
- **Channels:** 3Blue1Brown · StatQuest · DeepLearning.AI · Data Professor · Sentdex
- **Platforms:** [kaggle.com](https://kaggle.com) · freeCodeCamp ML certification

---

### 🔷 Semester 4 — Deep Learning & LLMs
**🎯 Goal:** Understand how neural networks actually work. Deploy something AI-powered.

- 📺 **Lectures:** Andrej Karpathy's *Neural Networks: Zero to Hero* — mandatory, watch every second
- 🔬 **Applied:** DeepLearning.AI LLM series + Nicholas Renotte / Sentdex for project work
- 🛠️ **Builds:** Vision project + basic NLP pipeline + a simple RAG system
- 🚀 **Deployment milestone:** At least one live AI application — public URL, clean documentation, GitHub repo
- **Channels:** Andrej Karpathy · DeepLearning.AI · Nicholas Renotte · Hugging Face · AssemblyAI
- **Platforms:** [huggingface.co](https://huggingface.co) · [kaggle.com](https://kaggle.com) · [cs50.harvard.edu/ai](https://cs50.harvard.edu/ai)

---

### 🔷 Semester 5 — Systems, Capstone & Interview Readiness
**🎯 Goal:** Go deep into how systems work. Build a capstone. Become interview-ready.

- 📺 **Lectures:** MIT OCW for OS/Networks/Systems + Low Level Learning / Computerphile for intuition
- 💪 **Daily:** LeetCode + mock interviews using takeUforward / Scaler content
- 🏁 **Capstone:** Full-stack + AI, deployed, with documentation that looks like serious open-source work
- 🎓 **Final milestone:** 400+ LeetCode problems · 1 deployed full-stack AI app · GitHub portfolio complete
- **Channels:** MIT OCW · Low Level Learning · Computerphile · Scaler · Ben Eater
- **Platforms:** [ocw.mit.edu](https://ocw.mit.edu) · [leetcode.com](https://leetcode.com) · [kaggle.com](https://kaggle.com)

## 🎯 Conclusion

Every ingredient for a rigorous CS and AI education is in this repository — CS50, MIT OCW, Stanford, freeCodeCamp, 3Blue1Brown, Karpathy, StatQuest, Striver. The full stack of serious free education exists here, and it costs nothing.

**The variable is not access. It was never access.**

The variable is whether you treat these resources like a real degree — with deadlines, problem sets, projects, failure, iteration, and shipped work — or like a playlist you scroll through when you feel motivated.

> 💬 *Build, don't watch. Struggle on problems. Ship real work. Use AI as a tool, not a crutch. Do that for 12–24 months and you won't just "know AI" — you'll be able to build almost anything you can clearly describe.*
